##🚗 Car Animation — HTML5 Canvas (Adobe Animate)
A lightweight HTML5 animation created in Adobe Animate CC, exported as HTML + JavaScript (CreateJS).
The project showcases a masked car animation displayed on a stylized background.

This project was created for educational and demonstration purposes only.
It is not used commercially, and all rights remain reserved.
